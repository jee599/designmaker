12:10 PM

This is likely a CORS preflight issue. Your OPTIONS handler probably isn't returning the right Access-Control-Allow-Methods header.

Check your middleware order — CORS needs to run before auth. Here's the fix:

app.use(cors()) // before auth
app.use(authMiddleware())
