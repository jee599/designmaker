01 — System

Grid System

12-column grid, 80px baseline. Every element snaps to intersection points. No "eyeballing" — the grid decides.

5 sizes. 2 weights. No exceptions. The constraint is the point — hierarchy comes from size and placement, not decoration.

Token System

Design tokens as source of truth. Change one value, every component updates. Tested across 847 component permutations.

07 — FAQ

Common questions

Can I use GRID with an existing design system?

Yes. GRID is additive. Import your existing tokens, map them to the grid structure, and the system enforces your rules. It doesn't replace your decisions — it enforces them.

What frameworks are supported?

React, Vue, Svelte, and vanilla HTML/CSS. Tokens export as CSS custom properties, SCSS variables, JS modules, or JSON. Framework adapters handle the rest.

How do I enforce the grid across a team?

Lint rules ship with every install. Components that break the grid throw warnings in dev mode. The Figma plugin snaps to grid automatically. Violations show up in the weekly design system health report.
