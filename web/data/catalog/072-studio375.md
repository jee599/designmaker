## Typography

- Body: Geometric Sans (400)

## Layout
- Hero: large typography with italic accent words
- Features: two-column asymmetric grid (left labels, right content)
- Cards: numbered project cards (01, 02, 03) with sequential ordering
- Buttons: minimal with hover underline animation
- Max width: max-w-6xl

## Key Details
- Italic accent words embedded within upright headings ("creative" studio)
- Numbered project cards with zero-padded sequential ordering (01, 02, 03...)
- Two-column asymmetric layout: narrow left column for labels/metadata, wide right for content
- Gold foil detail (#c9a96e) appears in case study imagery and accent strokes
- Scroll-triggered animated micro-interactions on section entry

## Section Order
Nav -> Hero Typography -> Featured Projects -> Services -> Awards -> Blog -> Contact -> Footer

## Background Pattern
#ffffff (hero) -> #f8f8f8 (projects) -> #ffffff (services) -> #f8f8f8 (awards) -> #ffffff (blog) -> #f8f8f8 (contact) -> #000000 (footer)

## Padding Pattern
hero: py-32 md:py-40 | projects: py-24 | services: py-20 | awards: py-16 | blog: py-20 | contact: py-24 | footer: py-16

## Material
Gold foil stamp on white stock paper

## Rhythm
- pacing: slow
- density: sparse

## Anti-patterns
- Do not mix italic and upright weights randomly — italic is used only for specific accent words within headings (typically the creative/expressive adjective), not for body copy
- Do not abandon the two-column asymmetric structure for centered layouts — the narrow label column / wide content column is the grid signature that carries the editorial identity

## Reconstruction Hints
- The asymmetric grid uses CSS Grid with columns: 180px 1fr (or 20% 80%) — left column holds short uppercase labels in small text, right column holds the actual content at full scale
- Numbered project cards prefix the project name with a span.number styled in the gold accent color and a larger font-size, using CSS counter or hard-coded zero-padded strings
